## About

slvelt.jsで作るModal  遊び
デザインは[CSSのみでお洒落なアニメーション付きのモーダルウィンドウを作ってみた](https://kuzlog.com/2018/01/21/2413/)から拝借



## 使い方

```
yarn install
```

の後、dist配下にJSが吐き出されます。  
index.htmlがサンプルです。


## 入れたモジュール

```
yarn add svelte svelte-loader webpack webpack-cli
```

## コンパイルコマンド

```
yarn run webpack
```
